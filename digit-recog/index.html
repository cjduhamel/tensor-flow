The first step was to create a model that we could run our training data agianst. This encompassed setting paramaiters, defineing layers, and calculating loss.

Training the Model

After fetching the training data, we reshape the image to a 2dVector which is then ran against our model. For every prediction loss is calculated and the process is reapeated. After 110 epochs or repitions of feeding batches of 30, The model is sufficently trained for use.

settings

Predicting

After the training the model(110 epochs or so) We then feed in one and one only example. This will then be predicted and checked for accuracy

The input is a handrawn digit in a 28 by 28 pixel grid. This is fed into our model and broken up into mulitple 2d vectors. The models job is essentailly to break the handdrawn digit into recognizable bits by recognizing which pixels are activated(white) and which are not.
With a total of 784 input nodes each pixle gets put through 2 hidden layers, in which weights functions are applied. After a prediciton is made, it gets compared to the correct answer and loss gets calculated.
This project is mostly made using mongoDB and EJS
